# Indicadores del Proyecto

## Proyecto

**Modernización de la Página Web y Plataforma Digital del Museo Infantil Trampolín**

## A) Indicador de gestión de avance

### 1. Porcentaje de avance de entregables del proyecto

**Definición:** mide el porcentaje de entregables completados en relación con el total de entregables planificados.

**Fórmula:**

```txt
(Entregables completados / Total de entregables planificados) x 100
```

**Utilidad:** permite dar seguimiento al progreso general del proyecto, identificar retrasos y tomar decisiones oportunas para mantener el cronograma.

## B) Indicadores de resultados

### 2. Porcentaje de funcionalidades prioritarias desarrolladas

**Definición:** mide cuántas funcionalidades principales del alcance han sido desarrolladas o implementadas.

**Fórmula:**

```txt
(Funcionalidades desarrolladas / Funcionalidades prioritarias definidas) x 100
```

**Funcionalidades consideradas:**

- Rediseño visual responsivo.
- Secciones informativas.
- Formulario digital para solicitudes educativas.
- Calendario de actividades o disponibilidad.
- Panel administrativo.
- Integración con WhatsApp e Instagram.
- Gestión básica de contenido.

**Utilidad:** permite verificar si el producto final cumple con los objetivos funcionales definidos para la primera versión.

### 3. Nivel de validación de requerimientos por stakeholders

**Definición:** mide el porcentaje de requerimientos revisados y validados por los stakeholders del proyecto.

**Fórmula:**

```txt
(Requerimientos validados / Total de requerimientos levantados) x 100
```

**Utilidad:** asegura que el proyecto avance alineado con las necesidades reales del museo y reduce retrabajos.

## C) Indicadores de impacto

### 4. Reducción del uso de canales manuales para solicitudes educativas

**Definición:** mide la disminución esperada del uso de llamadas, WhatsApp o correos para gestionar solicitudes educativas.

**Fórmula sugerida:**

```txt
((Solicitudes manuales antes - Solicitudes manuales después) / Solicitudes manuales antes) x 100
```

**Utilidad:** permite evaluar el impacto operativo de la plataforma al centralizar información y reducir carga manual.

### 5. Mejora en la experiencia digital del usuario

**Definición:** mide la percepción de mejora de usuarios al interactuar con la nueva web.

**Método:** encuesta de satisfacción con escala de 1 a 5.

**Criterios sugeridos:**

- Facilidad para encontrar información.
- Claridad de horarios, tarifas y actividades.
- Facilidad para solicitar visitas.
- Calidad visual.
- Accesibilidad móvil.

**Utilidad:** permite medir el valor percibido por familias, centros educativos y visitantes.
